# Get Started
```
cd ~
git clone https://github.com/enjoythecode/dot.git
dot/sh/link.sh
. ~/.bashrc
```

# Words
My setup. Efficiency of keystrokes (less friction) and portability (low high-tail friction)
are priorities.
